# Retail Data Warehouse & Power BI Analytics

## Overview
This project demonstrates the design and implementation of an end-to-end Retail Data Warehouse using PostgreSQL and Power BI.

The data pipeline follows the Medallion Architecture:

- Bronze Layer – Raw data ingestion
- Silver Layer – Data cleansing and transformation
- Gold Layer – Business-ready analytical datasets

A Star Schema was developed to support efficient reporting and business intelligence.

## Technologies Used
- PostgreSQL
- SQL
- Power BI
- Data Warehousing
- Star Schema
- ETL
- Data Modeling

## Data Architecture

Bronze → Silver → Gold → Power BI Dashboards

## Features
- Sales Performance Analysis
- Customer Analytics
- Product Performance Tracking
- KPI Monitoring
- Interactive Dashboards
